Это версия 1.0 приложения Pharmacy DataBase.

Для установки данного приложения необходимо установить сервер MySQL а также утилиту для работы с базами данных - MySQL Workbench.

Для этого установите:
1)mysql-installer-web-community-5.7.11.0
а затем:
2)mysql-workbench-community-6.3.6-win32
Далее либо позовите ответственного за внедрение данного ПО или если вы являетесь данным лицом, установите вручную базу данных, скрипт к которой хранится в файле SQL_query.txt.

Для работы приложению необходимы 2 библиотеки(Xceed.Wpf.Toolkit.dll, MySql.Data.dll) поэтому настоятельно рекомендуется не удалять данные файлы, а также файл Pharmacy DataBase.exe.config где хранится конфигурация данного приложения.

Также для демонстрации можно ввести тестировочную базу данных, в которой находятся некоторые значения(совпадения с реальными именами, товарами и пр. являются случайными), скрипт которой лежит в Insert.txt.

Внимание! Для установки необходимоого ПО нужен интернет.

Приложение можно запустить файлом Pharmacy DataBase.exe.

Если вы не являетесь ответственным по внедрению данного ПО, смотрите в интернете, как работать с утилитой MySQL Workbench для установки баз данных.

Приятного использования!
